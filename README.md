# PDFren でもっと豊かに!!

PDFren は、普段 PDF を開くときに感じる不満を解消するために作成した、Chrome 拡張機能型の PDF ビューアです。

Chrome 標準の PDF ビューアではやりにくい、**左右2画面表示・検索・LaTeXコピー・注釈・PDF作成・領域切り抜き・白紙PDFへの貼り付け** などを、1つのビューア内で扱えるようにすることを目的としています。

---

## 主な機能

### PDF閲覧

- PDFファイルの表示
- ページ番号を指定したジャンプ
- 拡大・縮小
- 表示エリア幅に合わせたズーム
- 左右独立した2画面表示
- 左右ペインの入れ替え
- 片側PDFの別ページを反対側に表示
- PDFを1枚だけ表示している時のフルスクリーン表示
- フルスクリーン中のキーボード操作
  - `Enter`：次ページ
  - `Backspace`：前ページ

### 検索・コピー

- PDF内テキスト検索
- 検索結果のハイライト
- 検索結果クリックによるページジャンプ
- 文字選択とコピー
- TextLayer のデバッグ表示切り替え

### 目次・アウトライン

- PDF内の見出し候補表示
- 左右ペイン別の目次表示
- 目次クリックによるページジャンプ
- 該当箇所のハイライト

### LaTeX / 数式補助

- PDF内の数式候補を検出
- 数式候補を LaTeX 形式で表示
- LaTeX を個別コピー
- 表示中ページの LaTeX をまとめてコピー
- 数式候補クリックによる該当箇所ジャンプ

### テキストボックス注釈

- PDF上にテキストボックスを追加
- テキスト入力
- テキストボックスの移動
- テキストボックスのサイズ変更
- テキストボックスの削除
- 文字サイズ変更
- 文字色変更
- ハイライト背景設定
- 太字
- 斜体
- 下線
- 左揃え / 中央揃え / 右揃え
- 書式クリア
- テキストボックス選択中は操作パネルを固定
- 操作パネルが開いた瞬間に PDF がずれて、クリック判定が外れて閉じる問題を抑制

### フリーハンド描画

- PDF上に手書き描画
- ペン色変更
- ペン幅変更
- 1本戻す
- 全消去
- ペインごとの描画保持
- 左右ペイン入れ替え時の描画情報入れ替え

### PDF作成機能

- 表示中PDFの反対側に白紙PDFを作成
- 白紙PDFにページ追加
- 現在ページを90度回転
- 白紙PDFを保存
- 注釈・描画・貼り付け画像を含めてPDFとして保存

### 領域選択・切り抜き貼り付け

- 元PDF上でドラッグして領域選択
- 選択範囲を白紙PDFへドラッグして貼り付け
- 選択範囲の四辺・四隅をドラッグしてトリミング
- トリミング用のガイド文言・ガイド図形は非表示
- 当たり判定だけを残した自然なトリミング操作
- PDF作成タブを離れたら領域選択を自動OFF
- 選択中の切り抜き状態を自動クリア
- 白紙PDFや貼り付け済み画像は維持

### 貼り付け画像の編集

- 白紙PDFへ貼り付けた画像を移動
- 貼り付け画像の四辺・四隅をドラッグして追加トリミング
- トリミング中に画像が縮小されない正確なプレビュー
- トリミング確定後に画像データへ反映
- 右下の青い丸ハンドルで拡大・縮小
- 貼り付け画像の削除
- 保存時に貼り付け画像もPDFへ反映

## 仕組み

### 1. PDFを自作ビューアで開く

ChromeでPDFを開くと、拡張機能がPDFのURLを検知します。

その後、Chrome標準のPDFビューアではなく、PDFren の自作PDFビューアへリダイレクトします。

```txt
PDFのURL
↓
Chrome拡張機能が検知
↓
PDFren のビューアURLへリダイレクト
↓
自作PDFビューアで表示
```

### 2. PDF表示と状態管理

PDFの表示状態は左右ペインごとに管理しています。

```txt
左ペイン
  - PDF URL
  - ページ番号
  - ズーム倍率
  - PDFバイト列
  - テキストボックス
  - 描画線
  - 貼り付け画像

右ペイン
  - PDF URL
  - ページ番号
  - ズーム倍率
  - PDFバイト列
  - テキストボックス
  - 描画線
  - 貼り付け画像
```

左右表示や入れ替え時にも、注釈・描画・PDFバイト列・白紙PDF状態をできるだけ維持するようにしています。

### 3. PDF作成と領域貼り付け

PDF作成タブでは、元PDFの反対側に白紙PDFを作成できます。

```txt
元PDFを表示
↓
白紙PDF作成
↓
元PDF上で領域選択
↓
必要に応じてトリミング
↓
白紙PDFへドラッグして貼り付け
↓
貼り付け後も移動・拡大縮小・追加トリミング
↓
保存
```

PDF作成タブを離れると、領域選択モードは自動でOFFになります。

### 4. 注釈付きPDF保存

保存時には、以下の内容をPDFへ反映します。

```txt
- テキストボックス
- フリーハンド描画
- 貼り付け画像
- トリミング済み画像
```

保存後は、新しいPDFとして読み込み直し、編集済み状態を反映したPDFを表示します。

## インストール方法

### 1. ReleaseブランチからZIPファイルをダウンロード

GitHubのReleaseブランチ、または配布されているZIPファイルをダウンロードします。

### 2. ZIPを解凍

ダウンロードしたZIPファイルを任意の場所に解凍します。

### 3. Chromeの拡張機能ページを開く

Chromeで以下を開きます。

```txt
chrome://extensions/
```

### 4. デベロッパーモードをON

右上にある **デベロッパーモード** をONにします。

### 5. 拡張機能を読み込む

**パッケージ化されていない拡張機能を読み込む** をクリックし、解凍したフォルダー内のビルド済みフォルダーを選択します。

例：

```txt
dist
```

または、配布形式によっては以下のようなフォルダーを選択します。

```txt
paper-viewer-v1.0
```

## 開発の進め方

Node.js と Git が入っていない場合は、先にインストールしてください。

### 1. Repositoryをclone

```bash
git clone https://github.com/Taiyo3901/Hackathon.v1.git
```

```bash
cd Hackathon.v1
```

### 2. 依存関係をインストール

```bash
npm install
```

### 3. 開発サーバーを起動

```bash
npm run dev
```

この実行は止めずに、別ターミナルまたは別タブで作業します。

### 4. ビルド

```bash
npm run build
```

### 5. 拡張機能用ファイルをdistへ反映

Windowsの場合：

```bash
copy extension\manifest.json dist\
```

```bash
copy extension\background.js dist\
```

macOS / Linux の場合：

```bash
cp extension/manifest.json dist/
```

```bash
cp extension/background.js dist/
```

### 6. Chromeへ読み込み

Chromeで以下を開きます。

```txt
chrome://extensions/
```

その後、

```txt
デベロッパーモード ON
↓
パッケージ化されていない拡張機能を読み込む
↓
dist フォルダーを選択
```

### 7. テスト

任意のPDFファイルをChromeで開きます。

PDFrenのビューアで開かれれば成功です。

## よく使う操作

### PDF表示

| 操作 | 内容 |
|---|---|
| ページ番号入力 | 指定ページへ移動 |
| ＋ | ズームイン |
| − | ズームアウト |
| ↔ | 表示エリア幅に合わせる |
| 左右表示 | 左PDFを右側にも開く |
| 入替 | 左右ペインを入れ替える |
| 全画面 | PDFのみ全画面表示 |

### 注釈

| 操作 | 内容 |
|---|---|
| テキスト | PDF上にテキストボックス追加 |
| 描画 | フリーハンド描画モード |
| 左保存 / 右保存 | 注釈付きPDFとして保存 |

### PDF作成

| 操作 | 内容 |
|---|---|
| 白紙PDF作成 | 反対側に白紙PDFを作る |
| ページ追加 | 白紙PDFにページを追加 |
| 現在ページを90度回転 | 白紙PDFの現在ページを回転 |
| 領域選択ON | 元PDFから範囲選択 |
| 選択範囲をクリア | 選択中の範囲を解除 |

---

## 注意点

- PDF作成機能は、基本的にPDFが1枚だけ表示されている状態で使うことを想定しています。
- 左右2画面表示中は、白紙PDF作成が制限されます。
- 白紙PDF上で画像をトリミングすると、削った部分は元に戻せません。
- 削りすぎた場合は、元PDFからもう一度領域選択して貼り付けてください。
- PDF作成タブを離れると、領域選択モードは自動的にOFFになります。
- 保存処理では、注釈をPDFに反映した新しいPDFとして読み込み直します。
